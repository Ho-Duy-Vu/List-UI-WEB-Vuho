# List-UI-WEB-Vuho

Tổng hợp các giao diện web thực hành HTML/CSS/JavaScript.

## Mô tả dự án

Repository này chứa nhiều mini UI web được tách theo từng thư mục (`web1` đến `web6`).
Mỗi phần là một giao diện độc lập để luyện bố cục, styling, responsive và tương tác cơ bản.

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Cấu trúc thư mục

```text
List-UI-WEB-Vuho/
|-- web1/
|   |-- asset/
|   |-- pages/
|   \-- style/
|-- web2/
|   |-- asset/
|   |-- pages/
|   \-- style/
|-- web3/
|   \-- web3/
|       |-- images/
|       |-- index.html
|       |-- main.js
|       \-- style.css
|-- web4/
|   |-- asset/
|   |-- index.html
|   |-- main.js
|   \-- style.css
|-- web5/
|   |-- asset/
|   |-- css/
|   |-- images/
|   |-- js/
|   \-- index.html
\-- web6/
	|-- index.html
	|-- script.js
	\-- style.css
```

## Cách chạy dự án

Vì đây là các trang tĩnh, bạn có thể chạy rất nhanh theo một trong các cách sau:

1. Mở trực tiếp file `index.html` hoặc file trong thư mục `pages` bằng trình duyệt.
2. Dùng extension Live Server trên VS Code để chạy và auto reload.
